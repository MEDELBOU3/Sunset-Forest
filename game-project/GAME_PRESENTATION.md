# Sunset Forest Survival Edition

## Introduction

Hello everyone,

Today I will present my game project called **Sunset Forest Survival Edition**.
This is a 3D survival game built with **JavaScript** and **Three.js**.
The player enters a dangerous forest, fights zombies, collects weapons and items, drives a vehicle, and tries to survive through different stages.

The goal of this project is not only to create a fun game, but also to practice game development concepts such as:

- 3D rendering
- character and enemy systems
- weapons and combat logic
- user interface design
- audio and atmosphere
- optimization and gameplay systems

## Topic 1: General Idea of the Game

The game takes place in a stylized forest environment during sunset.
The atmosphere mixes beauty and danger.
The player starts with basic equipment and must explore the map to find better weapons, ammunition, grenades, mushrooms, and useful upgrades.

At the same time, zombies attack the player in waves.
The player must survive, improve equipment, and continue progressing through the game.

## Topic 2: Core Gameplay

The gameplay is based on survival and action.
The player can:

- move freely in the 3D world
- aim and shoot with different weapons
- reload and manage ammo
- throw grenades
- loot weapons and supplies from the map
- use healing items like mushrooms
- switch weapons using a weapon wheel

Each weapon has different stats such as:

- damage
- fire rate
- ammo capacity
- zoom level
- scope compatibility

This makes the gameplay more varied and strategic.

## Topic 3: Enemies and Combat

One important part of the game is the zombie system.
Zombies chase the player, attack at close range, and create pressure during exploration and combat.

The game also includes:

- headshot detection
- damage numbers
- hit markers
- death effects
- wave progression

These features make combat more dynamic and satisfying.

## Topic 4: World, Loot, and Progression

The map contains many interactive elements.
The player can find:

- pistols
- rifles
- submachine guns
- sniper weapons
- grenades
- scope loot
- ammunition
- healing mushrooms

This loot system encourages exploration.
It also makes every session more interesting because the player becomes stronger over time.

The game also includes progression through stages and enemy waves, which gives the player a clear sense of challenge and advancement.

## Topic 5: Vehicle and Companion Features

Another interesting part of the game is that it is not limited to only walking and shooting.
The player can also use an armored vehicle, which adds variety to gameplay and movement.

In addition, the game includes a friendly companion character.
This companion follows the player, helps in combat, and adds more life to the game world.
It makes the experience feel closer to modern action-survival games.

## Topic 6: Visual Design and Audio

The visual style of the game is low-poly and stylized.
This gives the game a unique identity while also helping performance.

The game uses:

- sunset lighting
- fog and atmosphere
- weapon models
- particle effects
- minimap and HUD systems
- ambient sounds
- weapon sounds
- zombie sounds

Audio and visuals work together to create tension and immersion.

## Topic 7: Technical Side

From a technical point of view, this project helped develop many important programming skills.

The game includes systems such as:

- modular JavaScript sections
- 3D model loading
- animation loops
- collision handling
- weapon logic
- AI behavior
- HUD updates
- runtime optimization

This project is a good example of combining creativity and programming in one complete interactive experience.

## Conclusion

In conclusion, **Sunset Forest Survival Edition** is a 3D survival game that combines action, exploration, combat, and atmosphere.
It includes many features such as zombies, looting, weapons, grenades, scope systems, vehicles, and a friendly companion.

This project is important because it shows both game design ideas and technical implementation.
It represents a strong step in learning how to build a complete 3D game experience with JavaScript and Three.js.

Thank you for listening.
